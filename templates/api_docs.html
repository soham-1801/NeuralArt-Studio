API Documentation

Use the NeuralArt API to integrate style transfer into your applications.

Authentication

Include your API key in the Authorization header:

Authorization: Bearer nai_YOUR_API_KEY_HERE
Endpoints

POST /api/v1/transfer

Upload a content and style image to generate a stylized result.

Parameters (multipart/form-data):

  • content (file, required) - Content image (PNG/JPG)
  • style (file, required) - Style image (PNG/JPG)
  • alpha (float, optional) - Style strength 0.0-1.0 (default: 1.0)

cURL Example:

curl -X POST {{ url_for('api_transfer', _external=True) }} \
  -H "Authorization: Bearer nai_YOUR_API_KEY" \
  -F "content=@photo.jpg" \
  -F "style=@painting.jpg" \
  -F "alpha=0.8"

Python Example:

import requests

url = "{{ url_for('api_transfer', _external=True) }}"
headers = {"Authorization": "Bearer nai_YOUR_API_KEY"}
files = {
    "content": open("photo.jpg", "rb"),
    "style": open("painting.jpg", "rb"),
}
data = {"alpha": "0.8"}

resp = requests.post(url, headers=headers, files=files, data=data)
print(resp.json())
